Hello Fede!!

Congrats, you've just generated the UI Building Tools project.

If you see this, it means that you've already run "grunt" (or "grunt start").
All grunt tasks are defined and configured into the "Gruntfile.js" file in the root directory.
If you take a look at it, you can notice that there are already plugins used and tasks defined: "less", "requirejs", "jshint", "csslint", "connect" and "watch".

Exercises

What we ask you is to be able to create a very tiny web app using requirejs. Everything you need is alreay set up. Modify or add any file you need and don't hesitate installing a new plugin for grunt (or even for nodeJS) if it can help you.

Here are the exercise's requirements:

  1. modify the Gruntfile.js registering a "build" task regrouping all the "less", "requirejs", "jshint" and "csslint" tasks we want to execute when the app has started or by any changes triggered by the watch task
  2. create and add in the "TODO" section below a gallery of pure randomness. It should contain at least 10 items. Each item will be made of a random image, a random caption and a random author. Each of the random-thing should be a separate module (e.g.: image.js, author.js) called by you main file (main.js). You could also create a module in charge of the randomness (e.g.: random.js), which can be call by the other ones
  3. use hogan.js to support precompiled templates made with grunt-hogan and be able to generate html to inject into the web page
  4. add some styling into main.less to make it look like an exquisite piece of art
